# Image Generator Vue3

这是一个基于 Vue3 + Vite + Element Plus 构建的图片生成器。
后端项目未开源，暂不提供运行环境。
体验地址：huanst.cn

## 技术栈

- **前端框架:** Vue 3
- **UI 组件库:** Element Plus
- **构建工具:** Vite
- **HTTP 客户端:** axios

## 快速开始

### 安装依赖

```bash
npm install
```

### 运行开发环境

```bash
npm run dev
```

### 打包构建

```bash
npm run build
```
